# 科學研究常用工具 Scripts

本 repository 收錄一系列用於 **科學研究、數據分析與數值模擬** 的實用腳本（scripts）。  
這些工具主要用來簡化研究過程中的常見任務，例如資料處理、數值計算、科學圖表繪製與研究流程自動化。

所有腳本設計為 **簡單、輕量化且容易修改**，方便研究人員與學生快速整合到自己的研究流程中。

---

## 主要功能

本 repository 的 scripts 主要包含以下功能：

- 科學數據處理與分析
- 數值模擬與數值方法實作
- 科學圖表繪製（Scientific plotting）
- 實驗數據整理與檔案管理
- 研究流程自動化工具

---

## 使用環境

主要套件包含：

- numpy
- matplotlib
- scipy
- pandas

安裝方式：

```bash
pip install numpy matplotlib scipy pandas
```

---

## 使用方式

範例：執行數值模擬腳本

```bash
python advection_simulation.py
```
